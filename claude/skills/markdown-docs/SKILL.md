# Markdown writing

When writing or substantially editing long Markdown files, put each full sentence on its own physical line.

Do not wrap multiple sentences onto a single physical line.
Preserve normal Markdown structure: keep headings, lists, code blocks, tables, and front matter structurally valid.
Do not reflow unrelated Markdown content unless asked.
